## What to Do in Google Cloud Console

1. **Open:** https://console.cloud.google.com/apis/credentials?project=pacematch-gps

2. **Check if Android OAuth Client ID exists:**
   - Look for an entry with type: **"Android application"**
   - Package name: `com.pacematch.app`
   - SHA-1: `81:55:30:46:0C:B0:FC:A5:49:B8:96:F6:0B:CE:48:B6:B3:F9:ED:CD`

3. **If it doesn't exist, create it:**
   - Click **"Create Credentials"** (top of page)
   - Select **"OAuth client ID"**
   - Application type: **"Android"**
   - Package name: `com.pacematch.app`
   - SHA-1: `81:55:30:46:0C:B0:FC:A5:49:B8:96:F6:0B:CE:48:B6:B3:F9:ED:CD`
   - Click **"Create"**

4. **Wait 5-10 minutes** after creating (Google needs time to propagate)
